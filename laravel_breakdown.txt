--APP DIRECTORY
Core directory of the application
For the *models* files
	#CONSOLE
	Artisan commands
	#EXCEPTIONS
	Holds your laravel project's exception handling files, 
	which handles all the exceptions thrown by your Laravel project.
	#HTTP
	In here goes the different filters, requests and controllers.
	#PROVIDERS
	For different service providers

--BOOSTRAP
Contains all the bootstrapping scripts for the application

--CONFIG
All the configuration files

--DATABASE
For the database files

--RESOURCES
Sass files, language, files, templates...

--ROUTES
Contain all your definition files for routing such as console.php, api.php, channels.php, etc.

--PUBLIC
Contains the JS, CSS files, images, .htaccess, index, web.config..

--STORAGE
Session files, cache,compiled templates, misc...

--VENDOR
Composer and dependency files
